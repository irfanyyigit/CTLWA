# CTLWA - Commit to LinkedIn Web Application

**CTLWA**, yazılımcıların teknik gelişimlerini LinkedIn üzerinde profesyonel ve ilgi çekici bir şekilde paylaşmalarını sağlayan AI tabanlı bir web uygulamasıdır.

## Özellikler
- **GitHub Entegrasyonu:** Repo linki ile son commitleri otomatik çeker.
- **AI Gücü:** Google Gemini kullanarak teknik verileri hikayeleştirir.
- **Özelleştirilebilir:** Kullanılan teknoloji yığınına göre postu şekillendirir.

## Kurulum
1. Bu depoyu klonlayın: `git clone https://github.com/kullaniciadin/devstory.git`
2. Kütüphaneleri yükleyin: `pip install -r requirements.txt`
3. Uygulamayı çalıştırın: `streamlit run app.py`

## Gereksinimler
- GitHub Personal Access Token
- Google Gemini API Key (AI Studio)
